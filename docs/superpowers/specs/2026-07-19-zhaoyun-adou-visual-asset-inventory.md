# 《赵云与阿斗》视觉素材边界与 1:1 复刻验收清单

## 目标与证据口径

本文先回答“哪些位置确实有图案，哪些位置应保留文字”，再决定需要生成的素材。目标不是把整张截图贴成背景，而是在可交互 Canvas 中复刻其构图、字牌、工具、叠图和水墨反馈。

证据优先级如下：

1. [Apple App Store 官方页](https://apps.apple.com/cn/app/%E8%B5%B5%E4%BA%91%E4%B8%8E%E9%98%BF%E6%96%97/id6755426265)：官方明确说明“每一个汉字都是作战单位”“全程以汉字+水墨特效呈现战斗”，并列出赵云、关羽、张飞、黄忠等武将及其技能。
2. [Google Play 官方页](https://play.google.com/store/apps/details?hl=zh&id=com.zyyad.mihuan)：官方实机截图用于核对标题、棋盘、商店、武器装备和失败结算的实际外观。
3. [TapTap《赵云与阿斗》攻略区](https://www.taptap.cn/group/1271823/group-label/3322250)：社区实机图用于补齐主动/被动道具栏、普通铲、洛阳铲、招贤、陨石等当前版本细节；帖子对数值强弱的判断只作线索，不作为玩法真值。
4. 用户提供的 5 张战斗/标题参考图：作为本项目最终视觉验收基线，尤其用于核对高密度战斗、英雄光效、武器叠图和小程序竖屏布局。

## 核心结论

- 棋盘上的普通兵、英雄、敌兵主体和武器背包列表，本质上都以毛笔汉字为主，不应生成完整人物或完整武器缩略图取代文字。
- 真正需要图案的部分集中在：工具图标、战斗中的武器叠图、极小的敌兵冠饰/兵器、标题吉祥物、纸卷/木牌/云团，以及墨爆、火焰、箭雨、金色扇光等特效。
- 主动技能是圆形按钮，被动/道具是方形黑框按钮；两者不能混成一排同样的现代图标。
- 普通铲和洛阳铲不是两个并列的主动按钮：普通铲是进入五格营栏、由玩家拖到青色锁格消耗的物品；洛阳铲是底部方形被动槽，当前实现每 `60` 秒向有空位的营栏产出一把普通铲。
- 战斗操作行左侧的圆钮是毛笔/逆天改命笔，不是铲子；黄色“手持工具 + 宝箱”图标按现有实机语义更接近摸金/寻宝类被动，不能标成洛阳铲。
- 武器在“装备背包”页面是文字条目，在“棋盘战斗”中才以刀、枪、剑、弓等小图案叠到字牌上。这两个环节必须分别实现。
- 当前或后续若出现四个完整水墨敌军人物图，应判为偏离参考：普通敌军应退回“贼/兵/将或 Boss 姓名字形 + 血线 + 小装饰”的做法。

## 关键操作与精确表现

### 征兵

- 位置：营栏下方中央，是整个底部最宽的主按钮。
- 外观：橙褐/砖红矩形木牌，粗黑描边和下投影；中央大字“征兵”，下方或同区放白馒头图标与消耗数字。
- 状态：可用时砖红底；资源不足时转深灰但仍保留按钮轮廓和价格；点击后新字牌进入五格营栏。
- 不生成项：按钮文字、价格数字和兵牌内容均由 Canvas 绘制。

### 普通铲

- 外观：金黄铲头、黄褐短柄、粗黑手绘轮廓，位于五格营栏的浅色纸槽；参考实机中不是单纯一个“铲”字。
- 操作入口：它是营栏物品，不是左侧圆钮，也不是底部固定被动槽。按住营栏中的铲子拖到青色锁格；无效落点返回原营栏格。
- 操作反馈：拖起后可开地块出现亮黄描边；成功落下后锁地变成可部署地，同时营栏铲子和数量各减一。
- 与营栏关系：初始、征兵抽取或洛阳铲产出的普通铲都占用营栏槽；拖拽态必须保留完整铲形，不变成文本标签。

### 洛阳铲

- 外观：底部第一格方形被动槽中的黄色铲形图标；装备时使用金色框，右上显示剩余秒数或“待发”。它和营栏普通铲可以复用同类铲形资产，靠位置、框体和状态语义区分。
- 定位：它是“生产普通铲”的被动，不直接替玩家开地。当前规则固定为每 `60` 秒尝试向营栏生成一把普通铲。
- 满栏处理：营栏无空位时不吞掉本次产物，而是保留完成计时并显示“待发”；出现空格后立即补发。
- 操作验收：必须分别记录 `luoyangGenerated` 与 `shovelsUsed`，截图应包含“被动倒计时/待发 → 营栏出现普通铲 → 拖铲开地”三段状态。

### 毛笔/逆天改命笔

- 位置：征兵按钮左侧的圆形主动技能钮；参考中的圆盘轮廓和金色可用态保留，不能再把它解释为铲子入口。
- 当前操作：点击圆钮进入毛笔模式，再点击一个已部署的普通单位或英雄碎片；将其改写为当前关卡主推英雄尚缺的姓名字，消耗一支毛笔，并尝试完成英雄合成。
- 反馈：选中时圆盘变金色，棋盘可改写目标高亮；角落数量从 `×1` 变为 `×0`，同时记录 `brushUses`。

### 摸金/寻宝类图标

- 黄色手臂、工具与宝箱组合的方形图标更接近摸金/寻宝类被动，不是洛阳铲的复杂造型版本。
- 当前尚未用一手规则证据确认其正式名称和数值，因此只保留为未装备的灰暗状态图标；后续核实前不添加点击逻辑、收益或冷却。

### 英雄姓名牌

- 英雄仍然是“字”，不是人物卡：两个相邻白纸格分别写姓/名，例如“赵/云”“关/羽”“张/飞”“黄/忠”。
- 两格外围形成一体化彩色框与柔光，蓝、紫、金区分英雄/品质；字本身也可取同色，右上保留小等级数字。
- 合成/登场瞬间可出现更大的英雄名叠字和扇形金光，但动画结束后必须回到两个棋盘字牌。
- 武将范围以官方描述和实机证据为准；TapTap 提到“五虎上将加刘备”，不能据此一次性臆造所有未见技能。

### 武器叠图

- 武器是独立透明前景层，斜压在英雄或普通兵字牌上；可跨 1–3 格，但不能永久遮住姓名和等级。
- 参考色：剑刃青白、边缘红光；枪/矛以黑杆和银刃为主；弓箭为黑褐线条配橙黄箭迹；大招武器可加紫/金描边。
- 静态叠图、攻击轨迹和命中特效分层处理：静态武器留在牌面，轨迹只在攻击时出现，伤害数字处于最上层。
- 背包页面仍只显示装备名称文字，不能把战斗叠图复用成背包缩略图库。

### 敌兵

- 普通敌兵主体为黑色毛笔“贼/兵”等单字，红色水平血线紧贴其上；密集波次允许相互遮挡，形成乱墨感。
- 精英/Boss 可用黑红双字、姓名竖排或更粗墨块，并增加头冠、虫形、兵刃、烟气等小型图案；小图案面积不得压过主体汉字。
- 敌兵死亡用碎墨、淡出和“败”字/伤害数字表达，不出现写实尸体。
- 禁止项：写实古代兵卒、四类全身敌军立绘、统一等大的卡通士兵头像。

### 主动与被动技能

- 主动：战场主操作行左右两侧的圆形黑灰技能盘。左圆钮已确认为毛笔/逆天改命笔；可用时有金圈，耗尽时压暗并显示数量。右圆钮当前用于对局速度控制，不能仅凭外观写成未验证的技能名。
- 被动/道具：最底部一排方形深灰黑框格，当前首格洛阳铲为已装备被动，其余摸金/寻宝、符纸、招贤卷、陨石先以灰暗状态表达。普通铲不在这一排，它只存在于营栏及拖拽层。
- 商店/选择页：每项是浅宣纸横卡，左侧圆/方品质图标，中间为名称与说明，右侧黄色金币价格按钮；“我的道具”明确分成“主动/被动”两行。
- 形状约束：主动始终圆、被动始终方；任何未实现的技能不能仅因画面需要伪装成可点击按钮。

### 标题与结算

- 标题页：暖宣纸底上叠灰褐山石与中央亮区；红褐毛笔“赵云与阿斗”；军衔和星级在标题下；中部棋盘上有黑色“斗”与一个小型幽默兵卒；开始按钮为锈红木牌，顶部压两把交叉短剑。
- 标题页图标：头像、金币、齿轮、闪电、排行榜小卷轴、武器背包均是图案；数值、名称和提示条是文字。
- 结算页：浅蓝灰失败底或暖色胜利底；顶部不规则云团放结果字；中部黑色横卷轴放军衔和星；棋盘残局以“斗”和数个“贼”字呈现；底部是高对比关闭/继续按钮。
- 禁止项：结算页生成完整战斗插画、用现代玻璃拟态卡片替代云团/卷轴、把结果文字烘焙进不可修改位图。

### 工具图标图集

> 竖屏中国水墨小游戏的透明工具图标图集，十二个彼此完全分离的正面图标，严格四列三行：白色馒头、黄色短柄铲、黄色手臂持工具并配小宝箱的寻宝图标、黑杆毛笔、空白米色招贤卷、三枚橙色陨石、回收箭头与小兵器、黑色砚台、褐色泥地、灰黑尖刺、橙褐炸药袋、黄色符纸。暖色平涂，粗黑不规则手绘轮廓，轻微纸张颗粒，小尺寸 48px 仍清晰，统一光向，透明背景，无文字，无数字，无边框，无人物，不要 3D，不要写实，不要完整 UI。黄色铲表示铲类物品，手臂和宝箱组合必须是另一格寻宝图标，两者不可合并。

> 中国水墨手绘塔防游戏的透明武器叠图图集，青白剑刃红色护手的长剑、黑杆银刃长枪、黑褐弯弓、三支飞箭、厚背短刀，五件物品彼此分离，斜向动态构图，粗黑墨线，少量芥末金和紫色能量边缘，小尺寸高可读，透明背景，无文字，无人物，无场景，不要写实，不要 3D。

### 标题小兵吉祥物

> 原创竖屏水墨小游戏标题页角色素材：只画一个滑稽可爱的古代小兵，三头身，侧身向右疾跑，一只手把长枪斜举过肩，白色头巾，土黄色短甲，朱红腰带，灰蓝裤腿，表情坚定又憨萌。粗黑手绘墨线，暖米白宣纸配色，低饱和土黄和青绿，少量朱红点色，扁平 2D，小尺寸仍清楚。角色完整居中，四周大量留白，纯白单色背景，方便后续抠图。不要汉字、字母、标题、UI、边框、写实武将、3D、阴影、复杂背景、额外人物或马。

每次生成只做一类素材。若模型混入中文、边框或完整界面，下一轮直接沿用同一构图并加强“透明背景、无文字、目标数量完全分离、不可相互遮挡”；若连续三轮仍带标签，则改走去背景与审图裁切，不通过手工涂抹硬救错误字。
